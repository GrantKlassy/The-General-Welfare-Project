# Directives

This directory belongs to the user. It contains their ideas, plans, and direction for the project.

## Rules

- **Never create, edit, or delete any file in this directory.** This is enforced by a lefthook pre-commit hook, but do not rely on the hook — simply never touch `directives/`.
- Read files here when relevant to understand the user's intent.
- If you think something should be added or changed here, tell the user — don't do it yourself.
